Pharmashup

Mashup project for pharmacology researchers based on Linked Open Drug Data

Download this project as a .zip file Download this project as a tar.gz file

Don't Search, Just Find!

The main goal of this mashup project is providing an integrated interface for pharmacology researchers in order to enable them reach desired information about active ingredients and related topics easily.

This project is a part of a bachelor thesis which is running by a senior student at Ege University Department of Computer Engineering.

Project uses Linked Open Drug Data datasets such as DrugBank, DailyMed, DBPedia, LinkedCT...

There is also a planning to include Orangebook data.

This project also includes a SPARQL Query Builder in src/com/querymanager directory, which allows to create SPARQL query strings easily. Tutorial of this component will be available soon.

For any questions or suggestions, please feel free to contact.
